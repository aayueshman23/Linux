# Linux
Phase 1: Build the VM
Install a Linux VM, preferably Ubuntu Server or Rocky Linux.
Allocate enough resources for practice: 2 CPU, 2–4 GB RAM, and 20–40 GB disk.
Take a clean snapshot before you start.
Update the OS immediately after first boot. On Ubuntu use sudo apt update && sudo apt upgrade -y; on RHEL-based systems use sudo dnf update -y.
